## About Me

I'm a **Senior Software Engineer** based in **Lahore, Pakistan**, specializing in the **.NET stack** and full-stack web & mobile development. Currently at **Sintov**, I lead the **FiT** ecosystem — a comprehensive law firm management platform — and its mobile companion **FiTScan**, built with .NET 10, React Native, and AWS.

Previously at **VaporVM**, I architected **ManageX**, a modular monolithic platform for Fortune 500 enterprises using .NET Core, React, and MongoDB. I'm a **FAST NUCES** graduate (BSCS) and passionate about building reusable [**NuGet packages**](https://www.nuget.org/profiles/Shahoodzee) for the .NET community.
